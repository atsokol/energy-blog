# Warsaw bike-sharing trip flows

Warsaw's [Veturilo](https://veturilo.waw.pl/) bike-sharing system operated 384 stations across the city throughout the 2023 season (May–December). With millions of trips recorded, it offers a rare, granular view into how a European capital moves by bike.

This post maps the origin–destination flows between stations, letting you explore how trip patterns shift by time of day, day of the week, and month. Each arc connects a departure station to an arrival station; thickness and colour encode the number of trips.

## How Warsovians ride

The dominant pattern is radial: mornings pull trips inward toward the Old Town, the central business district, and major metro interchanges, while afternoons reverse the flow. On weekends the network loosens — flows spread to parks, the Vistula riverbanks, and leisure destinations in Praga. August peaks in absolute volume; October is already quieter as the season winds down.

### Net flow by station

Positive net flow (blue) means a station receives more trips than it dispatches in the selected period — a net destination. Negative (red) means more departures — a net origin. The top 25 stations by absolute net flow are shown.

```js
const netFlowData = (() => {
  const inbound  = d3.rollup(filteredFlows, v => d3.sum(v, d => +d.count), d => String(d.to));
  const outbound = d3.rollup(filteredFlows, v => d3.sum(v, d => +d.count), d => String(d.from));
  return stations.map(s => {
    const uid = String(s.uid);
    return {
      uid,
      lon: +s.lon,
      lat: +s.lat,
      net: (inbound.get(uid) ?? 0) - (outbound.get(uid) ?? 0),
    };
  });
})();
```

## Peak hours

Trip volume is highly concentrated in two windows: the morning commute (7–9 am) and the afternoon return (4–7 pm). The lunchtime shoulder is modest by comparison. Leisure riding after 8 pm is niche but visible near the Vistula on summer weekends.

```js
const hourlyTotals = d3.rollups(
  flowsByHour,
  v => d3.sum(v, d => +d.count),
  d => +d.hour
).map(([hour, total]) => ({ hour, total })).sort((a, b) => a.hour - b.hour);
```

```js
Plot.plot({
```

## Weekday vs weekend

Weekdays (Monday–Friday) account for around 70% of total trips. Saturday and Sunday show a notably flatter hourly distribution — the commute spike disappears and mid-afternoon sees the highest counts. The network also shifts geographically: leisure-oriented stations near parks and the Vistula waterfront dominate weekend net-destination rankings.

```js
const DOW_ORDER = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"];

const dowTotals = d3.rollups(
  flowsByDow,
  v => d3.sum(v, d => +d.count),
  d => d.day_of_week
).map(([dow, total]) => ({ dow, total }))
  .sort((a, b) => DOW_ORDER.indexOf(a.dow) - DOW_ORDER.indexOf(b.dow));
```

```js
Plot.plot({
  width: Math.min(width, 800),
  height: 240,
  x: { label: null, domain: DOW_ORDER },
  y: { grid: true, label: "Total trips" },
  color: {
    domain: ["Weekday", "Weekend"],
    range: ["#4e79a7", "#f28e2b"],
    legend: true,
  },
  marks: [
    Plot.barY(dowTotals, {
      x: "dow",
      y: "total",
```

## Seasonal variation

The season opens in May with moderate volume and peaks in August — Warsaw's warmest month. September remains strong, but October marks a sharp drop as temperatures fall and daylight shortens. The system closes for winter in late November.

```js
const MONTH_ORDER = ["May", "June", "July", "August", "September", "October", "November", "December"];

const monthTotals = d3.rollups(
  flowsByMonth,
  v => d3.sum(v, d => +d.count),
  d => d.month
).map(([month, total]) => ({ month, total }))
  .sort((a, b) => MONTH_ORDER.indexOf(a.month) - MONTH_ORDER.indexOf(b.month));
```

```js
Plot.plot({
```
